JavaScript project

The objective of this project is to create a web page that allows you to play a pixel war game using JavaScript.

  • HTML

  • CSS

  • JavaScript

  • VS Code

  • Swagger

Requirement Specification

  • JavaScript fundamentals: Understanding of functions, objects, DOM events, and the Fetch API.
  • API consumption: Ability to analyze Swagger documentation and perform HTTP requests (GET, POST, etc.).
  • User interface design: Build an interactive UI using HTML and CSS reflecting live game actions.
  • Client-side state management: Use localStorage to store the UID and reduce unnecessary re-entry of sensitive data.
  • Debugging and manual testing: Detect and fix issues related to network requests, display, or game logic.
  • Technical documentation: Provide JSDoc comments for each function and include a README with instructions and project details.

Acquired Skills

  • Dynamic DOM manipulation: Displaying and updating a 100x100 grid in real time.
  • API integration: Practical use of fetch for operations like searching, collecting, stealing coins, or hiring a spy.
  • Asynchronous JavaScript: Mastery of async/await to handle API calls efficiently.
  • Client-side security awareness: Securing UID handling with local storage and masked display.
  • User experience improvements: Designing additional features and enhancements beyond the reference implementation.
  • Clean code architecture: Well-structured JS files and reusable functions.
  • Version control & collaboration: Using GitLab to manage code versions and teamwork.

Simple Artificial Intelligence

University project Lowatem was constrcted with two phases: the second one was about AI , where we had to create a simple AI that can fight with another AI in a game named Lowatem.

  • Java

  • IntelliJ

Requirement Specification

  • Base of Java programming: Understanding Java syntax, data types, and control structures
  • JUnit testing: Creating and executing unit tests to ensure code quality
  • Debugging: Efficiently identifying and resolving programming bugs
  • Game Logic Implementation: Developing AI logic for the Lowatem game
  • Code Structure: Following Java naming conventions, indentation, and clean code practices
  • Design Document: Providing a document outlining algorithmic choices (e.g., graph traversal or shortest path strategies)

Acquired skills

  • AI Strategy Design: Creating logic to evaluate every possible action based on position and outcome
  • Performance Optimization: AI anticipates enemy moves to reduce losses and exploit opportunities
  • Comparative Analysis: Evaluation against other AIs like “Tâcheron” and “Ingénue” in level 9 rankings
  • Interface Work: Visual testing through the graphical mode of the Lowatem game
  • Collaboration: Phase 2 conducted in pairs, fostering teamwork and shared debugging

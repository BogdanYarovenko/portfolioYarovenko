Tower-defense project

IAcademyOfDoom is a single-player tower-defense and simulation game developed as a school project using C#. The game takes place on a futuristic university campus where waves of proto-AIs (artificial intelligences in training) navigate through the campus in order to reach their final exam room.

  • C#

  • UMLetino

  • Trello

  • VisualStudio

Requirement Specification

  • Understanding of object-oriented programming: Mastery of classes, inheritance, and polymorphism in Java.
  • Ability to analyze existing code: Interpret legacy structures to prepare for extension and modification.
  • Respect code conventions and structure: Follow naming standards, class architecture, and design patterns.
  • Use of UML tools: Model class diagrams and system interactions with UMLetino.
  • Algorithmic thinking: Design effective solutions for real-time simulation and pathfinding challenges.

Acquired Skills

  • Advanced Java development: Solid experience with OOP principles and implementation of complex logic.
  • Unit testing with JUnit: Designing and running test cases to ensure program correctness.
  • GUI development: Building and improving user interfaces using Java Swing and layout managers.
  • Code refactoring: Improving existing code structure while maintaining functionality.
  • Project collaboration tools: Using GitHub and Trello for version control and task management.
  • Documentation: Creating design documents detailing class structure, flow, and interactions.

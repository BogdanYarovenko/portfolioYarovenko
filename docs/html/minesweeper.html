Minesweeper

A java-based project that implements the classic Minesweeper game. The game features a simple user interface, allowing players to uncover tiles and flag mines while avoiding explosions.

  • Java

  • VisualStudio Code

Requirement Specification

  • Java basic skills requirement: Understanding of classes, methods, loops, arrays, and conditional logic.
  • Respect the structure: Code must follow an MVC (Model-View-Controller) structure for clarity and maintainability.
  • Program source code: Use consistent naming conventions, comments, and indentation for readability and teamwork.
  • Design documentation: Provide clear documentation of algorithmic logic, including game loop flow, tile state transitions, and user interactions.
  • Game rules implementation: Correctly implement Minesweeper mechanics, including mine generation, adjacency counting, and win/loss detection.

Acquired skills

  • Java programming: Developed strong object-oriented design patterns and efficient logic handling.
  • JUnit tests: Gained experience writing unit tests to validate game logic and core functions.
  • Simple UI: Built a responsive and intuitive interface using Java Swing components and event-driven programming.
  • Debugging: Improved troubleshooting skills by fixing UI rendering bugs and logic errors.
  • Algorithm design: Applied algorithms for safe tile uncovering and recursive area clearing.
  • Project documentation: Documented the development process and technical aspects effectively for future reference or team collaboration.
